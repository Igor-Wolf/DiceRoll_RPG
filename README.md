# diceroll_RPG

Description. 
The package diceroll_RPG is used to:
	- Roll dices with a random function
	- Get the result

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install diceroll

```bash
pip install diceroll_RPG
```

## Usage

```python
from diceroll_RPG import rolling
rolling.rollf()
```

```python
from diceroll_RPG import rolling
rolling.roll()
```
